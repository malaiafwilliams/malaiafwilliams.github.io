## CPSC 120 Favorite Labs

Lab 4 was one of my favorite assignments because it demonstrated how programming can be applied to practical, everyday situations. I enjoyed designing a program that converted milliliters into teaspoons, tablespoons, ounces, and cups using mathematical formulas. This lab strengthened my understanding of variables, functions, user input, and formatted output while improving my overall problem solving skills in C++.

### Lab 5

Lab 5 was one of my favorite labs because it introduced me to the importance of organizing code through functions and multiple source files. I enjoyed creating a program that calculated the area of a rectangle based on user provided dimensions. This assignment helped me develop a stronger understanding of modular programming, unit testing, and efficient code organization in C++.

### Lab 8

Lab 8 was one of my favorite labs because it connected programming concepts to a real world scoring system used in athletic competitions. I enjoyed developing a program that removed the highest and lowest scores before calculating the average of the remaining values. This lab improved my skills with vectors, command line arguments, loops, and writing functions to analyze data accurately.
