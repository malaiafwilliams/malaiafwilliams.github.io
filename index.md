## CPSC 120 Favorite Labs

### **Lab 4**

Lab 4 was one of my favorite assignments because it showed me how programming can be used to solve practical, everyday problems. I enjoyed creating a program that converted milliliters into teaspoons, tablespoons, ounces, and cups using mathematical formulas. Through this lab, I gained a stronger understanding of variables, functions, user input, and formatted output. It also helped me improve my attention to detail because even small calculation errors could affect the final result.

### **Lab 5**

Lab 5 was one of my favorite labs because it introduced me to the importance of writing organized and efficient code. I enjoyed developing a program that calculated the area of a rectangle based on user provided dimensions. This assignment taught me how functions and multiple source files work together in a larger project. It also strengthened my understanding of modular programming, unit testing, and the value of keeping code clear and well structured.

### **Lab 8**

Lab 8 was one of my favorite labs because it connected programming concepts to a real world scoring system used in athletic competitions. I enjoyed creating a program that removed the highest and lowest scores before calculating the average of the remaining values. This lab helped me build confidence working with vectors, loops, command line arguments, and functions that process data. It also showed me how programmers use logic to ensure fairness and accuracy in real applications.
